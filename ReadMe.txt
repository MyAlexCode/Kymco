Είναι μια σελίδα php με το css style που έφτιαξα για να γνωρίζω τις καταναλώσεις που κάνει ένα μηχανάκι ,
ανά χρονικό όριο την πλήρωση βενζίνης , και με αποτελέσματα ανά πλήρωση, και  συνολικά χιλιόμετρα ,
συνολική βενζίνη ,και συνολικά λεφτά ,που έχεις πληρώσει κατά την χρήση που του έχεις κάνει.

Η χρίση του είναι απλή, αφού εγκατασταθεί σε Php server φτιαχτεί η βάση δεδομένον "YourDB"
 με των πίνακα “YourTable” και τα πεδία id,saveDate, saveKmh, saveLitres, saveLitre_price
θα τρέξει και θα πρέπει να βάλεις τα χιλιόμετρα που έχει το μηχανάκι τα λίτρα βενζίνης
 που έβαλες για να το γεμίσεις και την τιμή του λίτρου, αυτό μετά την πρώτη εγγραφή 
δεν θα σου βγάλει κάτι εκτός από το πεδίο προηγούμενα χιλιόμετρα,
μετά κατά την χρίση θα προσθέτεις βενζίνη όσο θέλεις και αυτό
 θα σου βγάζει τα στατιστικά αυτόματα αφού έχεις περάσει της τρις τιμές που σου ζητάει σωστά
